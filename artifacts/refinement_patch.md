# Refinement Patch Artifact System

## Purpose

This document defines the refinement patch artifact used by the AI-native presentation cognition system.

The objective is NOT:
full presentation regeneration.

The objective IS:
localized cognition modification.

Traditional revision systems often behave as:

```text
feedback
↓
regenerate_slide
↓
replace_everything
```

This system behaves as:

```text
feedback
↓
cognition_failure_detection
↓
localized_patch_generation
↓
identity_preservation
↓
precision_refinement
```

The refinement patch exists to:

- preserve presentation continuity
- localize cognition modifications
- stabilize hierarchy behavior
- prevent unnecessary redesign
- support iterative collaboration

The refinement patch should behave like:

a cognition diff layer.

presentation identity.
