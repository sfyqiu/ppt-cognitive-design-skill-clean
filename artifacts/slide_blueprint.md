# Slide Blueprint Artifact System

## Purpose

This document defines the slide blueprint artifact used by the AI-native presentation cognition system.

The objective is NOT:
visual wireframing.

The objective IS:
slide-level cognition execution planning.

Traditional slide planning usually defines:

- layout templates
- visual placeholders
- static object arrangements

This system defines:

- cognition objectives
- hierarchy behavior
- reading flow
- mechanism emphasis
- semantic communication
- rendering intent

The slide blueprint exists to bridge:

```text
Cognition Planning
↓
Adaptive Rendering
```

The blueprint should describe:

how a slide should communicate.

---

# Core Slide Blueprint Philosophy

A slide blueprint is NOT:

a coordinate layout.

A slide blueprint IS:

a cognition execution structure.

The blueprint should answer:

```text
What should the audience notice?
↓
How should they interpret it?
↓
What should remain memorable?
```

The blueprint must preserve:

communication intent.
