# Cognition Plan Artifact System

## Purpose

This document defines the cognition plan artifact used by the AI-native presentation cognition system.

The objective is NOT:
presentation outlining.

The objective IS:
communication cognition planning.

Traditional presentation plans usually contain:

- slide titles
- section ordering
- topic summaries

This system produces:

- cognition objectives
- audience reasoning
- hierarchy strategies
- mechanism communication plans
- rendering intentions
- refinement preparation

The cognition plan exists to coordinate:

all downstream cognition execution.

---

# Core Cognition Plan Philosophy

A cognition plan is NOT:

a presentation outline.

A cognition plan IS:

a communication reasoning blueprint.

The plan should define:

```text
What the audience should understand
↓
How cognition should progress
↓
How hierarchy should behave
↓
How rendering should support understanding
↓
How refinement should remain stable
```

The plan should preserve:

communication intention.
