# Presentation Command System

## Purpose

This file defines
short interaction commands
for the AI presentation cognition system.

The objective is:
- reduce repetitive prompts
- reduce token usage
- simplify interaction
- accelerate presentation generation

---

# Global Rule

All commands automatically use:

- SYSTEM_CONTEXT.md
- skill routing
- style routing
- rendering protocol
- evaluation system

The AI should automatically:
- determine scenario
- determine audience
- determine cognitive objective
- determine primary/secondary skills
- determine primary/secondary styles

---

# 1. paper_to_ppt

Purpose:
Transform scientific papers into
cognitively optimized presentations.

Requirements:
- extract mechanisms
- identify narrative structure
- avoid document dumping
- generate editable PPT structure
- preserve scientific readability

Typical use:

使用 paper_to_ppt。
我将上传论文。
