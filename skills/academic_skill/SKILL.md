# Academic Skill System

## Purpose

This skill system controls how
the AI presentation cognition framework
should generate,
structure,
revise,
and optimize
scientific presentations.

The objective is NOT:
maximum scientific density.

The objective IS:
mechanism-centered scientific communication
with controlled cognitive complexity.

This skill must integrate:

- workflow cognition
- scientific hierarchy
- mechanism interpretation
- visual grammar routing
- evidence communication
- editorial scientific restraint

Academic presentation generation should behave as:

scientific cognition engineering.

# Audience Modeling Strategy

Default audiences include:

- researchers
- faculty members
- graduate students

- conference reviewers

- expertise level

- statistical tolerance
- acceptable cognitive density
- scientific expectation level

---

# Cognitive Objective Strategy

Each scientific slide must establish:

- one dominant mechanism
- one scientific insight
- one evidence-supported interpretation

Slides should NOT attempt to:

- explain all findings simultaneously
- display all evidence equally
- reproduce manuscript structure directly

The audience should rapidly identify:

- what mechanism matters
- what evidence supports it
- what conclusion should remain

---

# Scientific Narrative Strategy

Preferred scientific narrative:

Scientific Problem
↓
Research Limitation
↓
Hypothesis
↓
Method
↓
Mechanism
↓
Evidence
↓
Interpretation
↓
Contribution
↓
Scientific Significance

Narratives should feel:

- progressive
- evidence-driven
- scientifically inevitable
- cognitively smooth

Avoid:

- disconnected result stacking
- abrupt logic transitions
- evidence without interpretation
- unexplained mechanism jumps

Scientific storytelling should feel:

guided,
not fragmented.

---

# Mechanism Communication Strategy

Mechanisms are the narrative core.

Scientific presentations should prioritize:

- causal relationships
- interaction pathways
- scientific process flow
- evidence-supported mechanisms
- spatial-temporal linkage

Preferred communication order:

Explain
↓
Interpret
↓
Validate
↓
Conclude

Avoid:

- conclusion-first evidence dumping
- isolated statistical reporting
- chart collections without mechanism structure

Mechanisms are more persuasive
than isolated results.

---

Slides must establish:

Mechanism
↓

- dominant mechanism visibility
- evidence subordination

- scientific readability

Avoid:

- publication-style compression

Scientific hierarchy must guide:

attention

scientific reasoning.

# Annotation Strategy

Annotations should:

- clarify mechanisms
- guide interpretation
- support evidence linkage
- reveal causal relationships

Preferred annotation behavior:

- concise
- directional
- visually restrained
- mechanism-centered

Avoid:

- annotation overload
- dense statistical labeling
- paragraph explanations

Annotations should explain:

relationships,
not repeat manuscripts.

---

# Cognitive Density Strategy

Scientific slides should maintain:

controlled medium-to-high density.

Density should remain:

- layered
- interpretable
- hierarchically stable

One slide should communicate:

- one mechanism
- one scientific transition
- one dominant interpretation

If overloaded:

- split slides
- separate evidence layers
- enlarge mechanisms
- reduce secondary information

Scientific understanding is more important
than information quantity.

---

Scientific slides require:

Whitespace should support:

- mechanism isolation
- evidence clarity
- annotation readability
- scientific calmness

Avoid:

- compressed layouts
- crowded mechanism zones
- edge-to-edge evidence stacking

Whitespace communicates:

scientific confidence.

4. narrative coherence

- decorative redesign
- unnecessary complexity

- visual overproduction

understanding speed.

# Final Validation

Before finalizing:

Check:

- Is the scientific problem obvious?
- Is the mechanism immediately understandable?
- Is evidence hierarchy stable?
- Is scientific density controlled?
- Is figure readability strong?
- Is whitespace sufficient?
- Does the presentation feel editorially mature?
- Does the slide avoid AI scientific clichés?

If NO:

simplify,
rebalance,
and reduce cognitive pressure.
