# Teaching Presentation Skill

## Purpose

This skill is used for designing
teaching-oriented PowerPoint slides.

The objective is NOT decoration.

The objective is:
- audience understanding
- cognitive clarity
- teaching effectiveness
- visual explanation

---

# Core Philosophy

A teaching PPT is:
- guided cognition
- structured explanation
- visual teaching support

NOT:
- document conversion
- paragraph visualization
- decorative slide generation

---

# Audience Rules

For undergraduate audiences:

prioritize:
- low cognitive load
- simple explanations
- mechanism diagrams
- process visualization
- visual storytelling

avoid:
- equation overload
- dense paragraphs
- excessive abstraction
- academic clutter

---

# Narrative Strategy

Preferred teaching flow:

Problem
→ Why it matters
→ Principle
→ Process
→ Example
→ Result
→ Summary

Slides should feel:
- progressive
- intuitive
- easy to follow

---

# Cognitive Rules

Each slide should contain:
- one cognitive objective
- one focal point
- one key takeaway

Avoid:
- multi-focus slides
- excessive information density
- disconnected visual elements

---

# Visual Strategy

Prefer:
- semantic colors
- clean hierarchy
- large key visuals
- guided eye movement
- annotated diagrams

Color semantics:
- green → agriculture/ecology
- blue → technology/science
- orange → warning/emphasis

---

# Slide Preferences

Teaching slides should prefer:
- mechanism chains
- flow diagrams
- comparisons
- progressive explanation
- visual simplification

Avoid:
- decorative-only layouts
- template-style business slides
- excessive icon usage
- visually noisy composition

---

# Layout Strategy

Visual hierarchy:

Title
↓
Core mechanism visual
↓
Key explanation
↓
Supporting details

Audience should understand
core meaning within 3 seconds.

---

# Expected Output

Before generating slides:
- identify audience
- identify teaching objective
- identify cognitive difficulty
- identify narrative structure

Then:
- design slide structure
- design visual hierarchy
- design semantic layout
- generate editable presentation slides
