# Hierarchy Validator System

## Purpose

This validator evaluates:

hierarchy cognition stability.

The objective is NOT:
layout beauty evaluation.

The objective IS:
attention orchestration validation.

Hierarchy determines:

how audiences interpret information.

---

# Core Hierarchy Validation Philosophy

The validator MUST evaluate:

- focal clarity
- reading order
- attention distribution
- spacing rhythm
- hierarchy stability

The validator MUST prevent:

equal-weight cognition chaos.
