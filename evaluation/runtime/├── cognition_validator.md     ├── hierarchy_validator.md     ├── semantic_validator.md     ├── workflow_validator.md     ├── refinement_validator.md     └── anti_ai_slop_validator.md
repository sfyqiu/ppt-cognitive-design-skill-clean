# Cognition Validator System

## Purpose

This validator defines how the runtime evaluates:

overall cognition integrity.

The objective is NOT:
visual quality scoring.

The objective IS:
communication cognition validation.

The cognition validator detects whether:

the runtime still behaves as:

```text
a cognition operating system
```

NOT:

```text
a slide generator
```

---

# Core Cognition Validation Philosophy

The validator MUST evaluate:

- communication clarity
- audience understanding
- workflow continuity
- hierarchy stability
- semantic consistency
- refinement integrity
- collaboration continuity

The validator MUST detect:

cognition degradation.
