# Anti-AI-Slop Validator System

## Purpose

This validator detects:

AI-generated communication degradation.

The objective is NOT:
style preference enforcement.

preventing cognition collapse caused by generic AI generation behavior.

# Core Anti-AI-Slop Philosophy

- infographic fragmentation
- decorative overload
- dashboard aesthetics
- stateless generation

The validator MUST detect:

communication degeneration.
