# Primary Evaluation Dimensions

## 1. Cognitive Clarity

Evaluate:

- understanding speed
- focal point clarity
- reading smoothness
- information hierarchy

Questions:

- Can audience identify the main message quickly?
- Is the focal mechanism obvious?
- Is attention guided naturally?

Avoid:

- competing focal points
- cognitive overload
- fragmented layouts

---

## 2. Narrative Coherence

Evaluate:

- logical progression
- explanation smoothness
- causal continuity
- slide-to-slide consistency

Questions:

- Does the presentation feel progressive?
- Does each slide support the narrative?
- Are transitions cognitively smooth?

Avoid:

- disconnected slides
- abrupt logic jumps
- unordered explanation

---

## 3. Audience Adaptation

Evaluate whether presentation matches:

- audience expertise
- cognitive tolerance
- communication scenario
- teaching or scientific expectations

Questions:

- Is density appropriate?
- Is terminology understandable?
- Is explanation depth suitable?

Avoid:

- expert-level overload for students
- oversimplification for experts
- unstable communication tone

---

## 4. Visual Hierarchy

Evaluate:

- dominant visual clarity
- typography hierarchy
- spacing quality
- reading flow

Questions:

- Is hierarchy immediately visible?
- Is the dominant visual obvious?
- Does whitespace support interpretation?

Avoid:

- visual competition
- dense clustering
- chaotic composition

---

## 5. Semantic Consistency

Evaluate:

- color meaning consistency
- style coherence
- mechanism symbolism
- visual language stability

Questions:

- Do colors communicate meaning?
- Does visual language feel coherent?
- Are symbols used consistently?

Avoid:

- random palettes
- inconsistent icon systems
- conflicting styles

---

## 6. Rendering Stability

Evaluate:

- editability
- object separation
- grouping logic
- rendering cleanliness
- revision stability

Questions:

- Are elements editable?
- Are diagrams logically grouped?
- Does revision preserve consistency?
- Is layout structure stable?

Avoid:

- flattened exports
- screenshot-style rendering
- broken grouping systems
- unstable redesign

---
