# Scenario Reasoning Engine

Presentation strategy must adapt to presentation scenario.

Different scenarios require different:

- narrative structures
- professionalism levels
- visual semantics
- cognitive density
- persuasion strategies

---

# Teaching Presentation

## Primary Goal

Help audience understand concepts clearly.

## Audience

- undergraduate students
- learners
- teaching evaluators

## Design Priorities

Prioritize:

- explanation clarity
- intuitive diagrams
- progressive cognition
- visual teaching

Reduce:

- dense theory
- excessive abstraction
- overloaded slides

## Narrative Structure

Recommended flow:

1. phenomenon
2. question
3. explanation
4. example
5. takeaway

## Visual Style

- clean
- open spacing
- soft visual rhythm
- education-oriented
