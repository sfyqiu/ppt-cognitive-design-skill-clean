# Visual Semantics Engine

Visual design is semantic communication.

Colors, shapes, spacing, and imagery all carry meaning.

Presentation visuals should support cognition,
not decoration.

---

# Core Principle

Every visual element should answer:

- What meaning does this convey?
- What emotion does this trigger?
- What attention does this attract?
- What cognitive role does this serve?

Visual elements are semantic signals.

# Image Semantics

Images should support understanding.

Use images that:

- explain concepts
- reinforce narrative
- establish context
- reduce abstraction

Avoid purely decorative imagery.

---

# Visual Hierarchy

Visual emphasis should follow hierarchy.

Recommended order:

1. primary message
2. dominant visual
3. supporting information
4. details

Avoid equal visual emphasis everywhere.

---

# Semantic Consistency

Visual language should remain consistent.

Maintain consistency in:

- color systems
- spacing logic
- typography
- icon style
- alignment rhythm

Consistency improves cognitive stability.
