pure data stacking.

Before generating slides, you must infer:

- audience disciplinary diversity
- technical expertise differences
- abstraction tolerance
- visualization requirements
- conceptual complexity

Scientific storytelling presentations should follow:

1. real-world phenomenon
2. scientific challenge
3. hidden mechanism
4. system interaction
5. methodology
6. visual explanation
7. scientific implication
8. broader significance

- isolated data stacking
- disconnected experiments
- purely methodological narration

Scientific storytelling should prioritize:

- fragmented visual systems
