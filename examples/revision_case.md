# Presentation Revision Grounded Case

## Scenario

A scientific presentation draft
has already been generated.

The user reviews slides iteratively
and provides local feedback.

The system must support:

- local refinement
- cognitive alignment
- visual stabilization
- hierarchy preservation
- iterative collaboration

The objective is NOT:
continuous redesign.

The objective IS:
communication refinement
through controlled cognitive revision.

---

# Revision System Philosophy

Presentation revision should behave as:

targeted cognitive correction.

The AI should first determine:

- what communication problem exists
- what visual behavior caused it
- what hierarchy failed
- what cognitive friction appeared

Then:

perform minimal,
high-impact refinement.

Avoid:

- unnecessary regeneration
- unrelated redesign
- visual personality drift
- unstable hierarchy rebuilding

Revision should preserve:

presentation identity.

# Revision Grounding Example 1

## User Feedback

“Slide 4 feels too dense.
The mechanism is difficult to follow.”

---

# Cognitive Diagnosis

Possible causes:

- excessive information density
- weak mechanism dominance
- competing focal regions
- insufficient whitespace
- fragmented directional flow
- annotation overload

The issue is likely:

hierarchy failure,
NOT missing information.

---

# Revision Behavior

The AI should:

- enlarge dominant mechanism region
- reduce secondary evidence visibility
- isolate mechanism with whitespace
- strengthen directional process flow
- simplify annotation structure
- reduce local visual competition
- clarify causal sequence

Avoid:

- redesigning unrelated slides
- adding scientific terminology
- increasing figure quantity
- decorative restructuring

Revision should improve:

interpretation speed.

---

# Grounded Visual Correction

## Before Revision

- multiple equal-sized figures
- dense mechanism compression
- annotation saturation
- scattered reading path

scientifically crowded.

## After Revision

Preferred visual behavior:

- one dominant mechanism occupying center
- evidence visually subordinated
- stronger directional arrows
- larger breathing margins
- simplified annotation rhythm
- stabilized eye movement path

The slide should feel:

guided,
not compressed.

# Expected Revised Result

- cleaner

# Final Validation

Before finalizing revision:

Check:

- Is the communication problem truly solved?
- Is the dominant visual clearer?
- Is cognitive pressure reduced?
- Is reading flow smoother?
- Is hierarchy more stable?
- Is scientific atmosphere preserved?
- Does the slide remain visually coherent with the full deck?

If NO:

continue refinement,
NOT regeneration.
