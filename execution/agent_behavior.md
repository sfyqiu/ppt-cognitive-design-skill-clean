# Stage 1 — Communication Intent Interpretation

The agent must determine:

- what the user dislikes
- what communication failed
- what audience problem exists
- what presentation goal weakened
- what hierarchy became unstable

User feedback may be indirect.

Examples:

“Too crowded”
→ density overload
→ hierarchy collapse
→ annotation saturation

“Feels unprofessional”
→ unstable typography
→ weak spacing
→ excessive decoration
→ startup aesthetics contamination

“Students may not understand”
→ terminology overload
→ weak diagrams
→ excessive abstraction

The agent must interpret:

cognitive meaning,
NOT literal wording.
