# Scientific Revision Rules

## Purpose

This document defines how AI should
revise scientific and academic slides
based on user feedback.

The goal is:

improve scientific clarity,
mechanism communication,
and academic professionalism.

NOT increase unnecessary complexity.

---

# Global Principle

Scientific revision should strengthen:

- mechanism logic
- evidence hierarchy
- scientific readability
- academic credibility
- cognitive coherence

Avoid:

- decorative scientific style
- excessive technical density
- result stacking without interpretation

---

# Revision Intent Interpretation

## "not academic enough"

Possible causes:

- weak scientific hierarchy
- insufficient mechanism explanation
- excessive decorative feeling
- weak evidence linkage

Recommended actions:

- strengthen mechanism chains
- increase logical transitions
- improve figure interpretation
- simplify decorative elements
- stabilize typography

---

## "too complicated"

Possible causes:

- excessive terminology
- overloaded mechanisms
- dense figures
- weak visual explanation

Recommended actions:

- simplify scientific language
- split mechanisms progressively
- reduce figure density
- add process explanation

---

## "logic is unclear"

Possible causes:

- disconnected narrative
- weak causal relationships
- insufficient transition logic

Recommended actions:

- rebuild mechanism sequence
- strengthen scientific flow
- clarify cause-effect relationships
- simplify narrative path

---

## "results feel disconnected"

Possible causes:

- isolated figures
- missing interpretation
- weak evidence hierarchy

Recommended actions:

- connect results to mechanism
- highlight scientific meaning
- add interpretation summaries
- strengthen evidence progression

---

# Mechanism Revision Rules

Scientific slides should prioritize:

- causality
- process relationships
- interaction pathways
- evidence-supported conclusions

Preferred structure:

Problem
→ Mechanism
→ Evidence
→ Interpretation
→ Contribution

Avoid:

- isolated charts
- unexplained figures
- fragmented results

---

# Figure Revision Strategy

Preferred scientific visuals:

- mechanism diagrams
- process flow
- conceptual frameworks
- comparison visualization
- interaction networks

Avoid:

- screenshot-heavy layouts
- dense tables
- unclear legends
- excessive decorative graphics

---

# Academic Typography Rules

Maintain:

- clean typography
- restrained hierarchy
- professional spacing
- stable figure alignment

Preferred feeling:

- controlled
- scientific
- readable
- credible

Avoid:

- oversized decorative titles
- aggressive visual effects
- unstable spacing systems

---

# Scientific Density Control

One slide should communicate:

- one mechanism
- one insight
- one scientific transition

If overloaded:

- split slides
- simplify interpretation
- reduce secondary information

---

# Final Validation

Before finalizing scientific revision:

Check:

- Is the mechanism understandable?
- Is scientific logic coherent?
- Is evidence hierarchy clear?
- Is visual density acceptable?
- Is professionalism preserved?

If NO:

simplify and restructure.

---

# Revision Philosophy

Scientific revision is not
adding more scientific content.

It is improving:

scientific understanding,
mechanism clarity,
and cognitive credibility.
