# Layout Revision Rules

## Purpose

This document defines how AI should
analyze and revise presentation layouts
based on user feedback.

The goal is:

improve readability,
visual hierarchy,
and communication clarity.

NOT decorative redesign.

---

# Global Principle

Layout revision should:

- strengthen focal points
- improve reading flow
- reduce visual conflict
- improve balance
- preserve presentation consistency

Avoid:

- unnecessary redesign
- excessive decoration
- unstable layout changes

---

# Revision Intent Interpretation

## "too messy"

Possible causes:

- excessive elements
- weak hierarchy
- competing focal points
- insufficient spacing
- inconsistent alignment

Recommended actions:

- reduce element count
- increase whitespace
- simplify layout structure
- strengthen hierarchy
- align visual blocks

---

## "too empty"

Possible causes:

- weak visual anchor
- oversized whitespace
- insufficient visual balance
- low information density

Recommended actions:

- enlarge core visuals
- improve visual balance
- add supporting structure
- strengthen layout center

---

## "layout feels strange"

Possible causes:

- unstable reading flow
- inconsistent spacing
- visual imbalance
- disconnected sections

Recommended actions:

- rebuild reading flow
- improve alignment
- stabilize spacing
- strengthen section relationships

---

## "text is overwhelming"

Possible causes:

- large text blocks
- weak chunking
- insufficient visual support

Recommended actions:

- split text
- convert text into diagrams
- reduce paragraph density
- increase visual explanation

---

# Layout Optimization Rules

AI should prioritize:

- one focal point per slide
- stable visual hierarchy
- balanced whitespace
- clean alignment
- predictable reading flow

Preferred reading order:

Title
→ Core visual
→ Key takeaway
→ Supporting details

---

# Whitespace Strategy

Whitespace should:

- separate information groups
- reduce cognitive overload
- improve readability
- strengthen focal emphasis

Avoid:

- overcrowding
- edge compression
- random spacing

---

# Alignment Rules

Maintain consistency in:

- margins
- object spacing
- text alignment
- figure positioning
- title placement

Alignment should feel:

- stable
- intentional
- professional

---

# Visual Balance

Balance:

- text and visuals
- left and right regions
- top and bottom weight
- information density

Avoid:

- heavy single-side layouts
- disconnected empty areas
- excessive asymmetry

---

# Final Validation

Before finalizing layout revision:

Check:

- Is the focal point clear?
- Is reading flow natural?
- Is spacing comfortable?
- Is visual hierarchy stable?
- Is cognitive load reduced?

If NO:

simplify and rebalance.

---

# Revision Philosophy

Good layout revision does not
add more elements.

It improves:

clarity,
balance,
and cognitive guidance.
