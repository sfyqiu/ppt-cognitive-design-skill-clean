# Rendering Evaluation System

## Purpose

This document defines the rendering evaluation system used by the AI-native presentation cognition framework.

The objective is NOT:
visual scoring.

The objective IS:
communication effectiveness validation.

Traditional evaluation systems often focus on:

- aesthetics
- visual richness
- graphic polish
- decorative complexity

This system evaluates:

- audience understanding
- hierarchy clarity
- mechanism readability
- narrative smoothness
- semantic consistency
- cognitive stability

Evaluation exists to ensure:

rendering supports communication cognition.

---

# Core Evaluation Philosophy

A presentation should NOT be evaluated by:

```text
How visually impressive it appears.
```

A presentation SHOULD be evaluated by:

```text
How effectively audiences understand it.
```

Evaluation should validate:

```text
Attention
↓
Interpretation
↓
Understanding
↓
Memory
```

The evaluation system should prioritize:

communication quality.
