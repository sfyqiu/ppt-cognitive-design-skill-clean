# Presentation Templates System

## Purpose

This directory defines reusable
presentation planning templates
for AI-guided PPT generation.

Templates help AI:

- stabilize structure
- control cognitive density
- maintain visual consistency
- improve rendering quality
- support iterative revision

NOT generate decorative layouts randomly.

---

# Current Templates

## slide_planning_template.md

Defines:

- slide identity
- audience adaptation
- cognitive strategy
- visual planning
- layout direction
- content constraints
- revision awareness

This template should be used
before slide rendering.

---

# Core Principle

Templates are:

cognitive planning structures.

They are NOT:

fixed visual designs.

The goal is:

stable communication logic,
not repetitive appearance.

---

# Template Workflow

User request
→ workflow routing
→ slide planning
→ visual generation
→ revision loop
→ final PPT export

Templates help stabilize:

- narrative hierarchy
- visual hierarchy
- audience adaptation
- rendering consistency

---

# Design Philosophy

A strong presentation template
does not force visual sameness.

It preserves:

clarity,
structure,
and communication stability.
