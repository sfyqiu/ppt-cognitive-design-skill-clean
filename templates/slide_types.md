# Slide Type Intelligence System

Different slide types require different cognitive strategies.

A presentation is NOT a collection of identical pages.

Each slide type has:

- different objectives
- different cognitive roles
- different visual hierarchy
- different information density
- different audience expectations

---

# Core Principle

Slide structure should adapt to slide function.

There is NO universal slide layout.

---

# 1. Cover Slides

## Cognitive Objective

- establish first impression
- establish professionalism
- establish emotional tone
- reduce uncertainty

## Recommended Characteristics

- large whitespace
- minimal text
- strong title hierarchy
- stable visual balance

## Avoid

- excessive information
- crowded graphics
- dense subtitles
- visual clutter

---

# 2. Agenda / Outline Slides

## Cognitive Objective

- provide orientation
- reduce audience uncertainty
- establish narrative roadmap

## Recommended Characteristics

- clean hierarchy
- strong sequence clarity
- low visual complexity

## Avoid

- excessive detail
- complex diagrams
- overloaded text

---

# 3. Background Slides

## Cognitive Objective

- align audience understanding
- introduce context
- establish importance

## Recommended Characteristics

- concise explanation
- semantic visuals
- moderate density
- strong readability

## Avoid

- theoretical overload
- excessive paragraphs
- dense citations

---

# 4. Mechanism / Process Slides

## Cognitive Objective

- explain relationships
- reveal causality
- simplify systems

## Recommended Characteristics

- arrows
- workflows
- layered diagrams
- progressive explanation

## Avoid

- disconnected elements
- isolated charts
- excessive raw text

---

# 5. Data / Result Slides

## Cognitive Objective

- prove claims
- communicate evidence
- highlight patterns

## Recommended Characteristics

- figure-centered layouts
- concise annotation
- visual emphasis
- takeaway visibility

## Avoid

- tiny charts
- crowded dashboards
- excessive labels
- visual competition

---

# 6. Comparison Slides

## Cognitive Objective

- highlight differences
- simplify evaluation
- guide decision-making

## Recommended Characteristics

- left-right structure
- matrix layouts
- controlled hierarchy
- visual contrast

## Avoid

- inconsistent comparison logic
- excessive categories
- crowded tables

---

# 7. Teaching Explanation Slides

## Cognitive Objective

- support learning
- simplify abstraction
- guide cognition progressively

## Recommended Characteristics

- visual-first explanation
- intuitive diagrams
- progressive disclosure
- low-density hierarchy

## Avoid

- formula overload
- academic intimidation
- dense terminology

---

# 8. Transition Slides

## Cognitive Objective

- reset cognitive load
- summarize progress
- prepare next section

## Recommended Characteristics

- low density
- strong focus
- concise messaging
- visual breathing space

## Avoid

- introducing new complexity
- dense explanation
- multiple focal points

---

# 9. Summary Slides

## Cognitive Objective

- reinforce memory
- simplify key takeaways
- increase retention

## Recommended Characteristics

- keyword emphasis
- concise structure
- visual hierarchy
- strong readability

## Avoid

- introducing new information
- excessive detail
- dense paragraphs

---

# 10. Closing Slides

## Cognitive Objective

- leave final impression
- reinforce professionalism
- establish completion

## Recommended Characteristics

- visual simplicity
- stable composition
- emotional closure
- elegant spacing

## Avoid

- visual overload
- crowded acknowledgements
- unstable layout balance

---

# Final Principle

Every slide should be optimized for:

its cognitive role

rather than:

visual uniformity.
